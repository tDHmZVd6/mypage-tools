【注意事項】
※※何をするコードか理解できない人は実行しないでください。※※

・コード内に記述されたURLのドメイン部分はマスクしてあるので、実行前に正しいURLに修正してください。
・エラー処理とか一切無いんで、必要なら自分で追加してください。
・使用する場合は自己責任でお願いします。

【設定ファイル】
・login_info：IDとパスワードを記述するファイル
※スクリプトと同じフォルダに置いてください

【スクリプト】
・confirm.py：実行するとアサインを確認状態にする(確認ボタンを押した状態にする)
・Kyuuka.py：コード内に記述された日付を、一括で勤務不可に設定する
・set_schedule.py：翌月の勤務可、不可を一括で設定する(月～金を勤務可に、土日を勤務不可に設定する。祝日の勤務不可設定は未対応)
(それぞれ独立したスクリプトです)
※スクリプト内に変数を直書きしてあるものは、直接スクリプト内の変数を修正してから実行してください。
